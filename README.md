# 📂 My Developer Portfolio

A dynamic and responsive frontend portfolio website that showcases my professional programming projects and web applications. Built as part of the **Udemy 100 Days of Code: The Complete Python Pro Bootcamp (Day 83)**.

## 🌐 Live Demo

**Visit the live portfolio:** [https://StamatisKamisakis.github.io/my-portfolio/](https://StamatisKamisakis.github.io/my-portfolio/)

> ⚡ **Fast & Responsive:** Hosted directly on GitHub Pages for blazing-fast loading speeds and 100% uptime.

---

## 📋 Project Overview

This application acts as a centralized hub for all my development work. It dynamically fetches project data (titles, descriptions, and production links) from a structural `projects.json` file and renders them into clean, interactive user-interface cards using asynchronous JavaScript.

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Data Management:** JSON (JavaScript Object Notation)
- **Deployment & Hosting:** GitHub Pages
- **Version Control:** Git & GitHub

## 📂 Repository Structure

- `index.html` - The core structural layout and styling of the portfolio.
- `script.js` - Dynamic DOM manipulation and asynchronous API fetch operations.
- `projects.json` - The database containing project metadata and live links.
- `.nojekyll` - Ensures GitHub Pages bypasses Jekyll processing for direct asset loading.

## 🚀 Future Additions

As I progress through advanced tracks, newly deployed web services (Flask/PostgreSQL on Render, Full-Stack applications, etc.) will be appended directly into `projects.json` to keep this portfolio continuously updated.
